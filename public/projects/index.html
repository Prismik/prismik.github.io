SwiftTracer - A physically based rendering engine

SwiftTracer is a Swift implementation of a physically based rendering engine inspired by PBRT, Mitsuba and many other contributors. It was built to support my research efforts as part of my master’s degree. The engine currently supports several image formats like png, jpg, pfm and exr. See for yourself on github.
